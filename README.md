# Overview

This repository shows the efforts that I made into making an app in Andriod Studio. The app is a basic graduation planner that helps college seniors identify what they
still need to complete before graduation, make plans for post graduation, and make daily tasks to accomplish their goals. Finally, the user can export and print off the 
file to see their goals and tasks on a non-digital platform.


Here is a breif walkthrough of my code: https://www.youtube.com/watch?v=xm26ouCttQ0

# Development Environment

For this project, I used Android Studio and mainly worked within the Kotlin language. I also used a little bit of Java to increase functionality. Finally, within
this project, I expereimented using the Android Studio Assitant to help organize files and proofread my code.
